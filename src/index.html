Портфолио

Cafe Burger House

Одностраничный сайт (landing page) бургерной - Burger House. Сайт адаптирован для различных разрешений экранов. Разделы "Фирменные блюда" и "Меню" реализованы с помощью слайдера Swiper; фотогалерея сделана с помощью лайтбокса Tobii; добавлен фрагмент карты. Проект собирается с помощью Webpack.

News-Analyzer

Сервис, позволяющий находить все статьи по запросу за последнюю неделю и анализировать на диаграмме популярность темы. Сервис работает c API newsapi.org и github.com. Блок "История коммитов из GitHub" реализован с помощью слайдера Swiper. Проект собирается с помощью Webpack.

Mesto
